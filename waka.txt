echo 'source ~/.bash-wakatime/bash-wakatime.sh' >> ~/.bashrc

source ~/.bashrc

# 1. 터미널 WakaTime 도구 설치
python3 -c "$(wget -q -O - https://raw.githubusercontent.com)"
2. 라즈베리파이 터미널 WakaTime 설정 
라즈베리파이 터미널에서 다음 명령어를 실행하여 terminal-wakatime을 설치합니다. 
bash
# 1. 터미널 WakaTime 도구 설치
python3 -c "$(wget -q -O - https://raw.githubusercontent.com)"
설치 도중 API Key를 물어보면 복사한 API Key를 붙여넣습니다. 
3. API Key 수동 설정 (선택 사항)
설치 중 키를 입력하지 않았거나, 나중에 변경하려면 ~/.wakatime.cfg 파일을 수정합니다. 
파일 열기: nano ~/.wakatime.cfg
내용 수정/추가:
ini
[settings]
api_key = YOUR_WAKATIME_API_KEY
Ctrl + X, Y, Enter를 눌러 저장하고 나옵니다. 
4. 쉘(Shell) 설정 파일 수정
설치 스크립트가 자동으로 ~/.bashrc 또는 ~/.zshrc를 수정하지 않은 경우, 직접 추가해야 합니다. 
Bash 사용 시 (기본): nano ~/.bashrc
Zsh 사용 시: nano ~/.zshrc
파일 맨 아래에 다음 내용을 추가합니다.
bash
# Wakatime Terminal Tracking
if [ -f ~/.wakatime/terminal-wakatime.sh ]; then
    . ~/.wakatime/terminal-wakatime.sh
fi
5. 적용 및 확인
설정을 적용하기 위해 터미널을 다시 시작하거나 다음 명령어를 실행합니다. 
bash
source ~/.bashrc  # 또는 source ~/.zshrc
터미널에서 명령어를 몇 개 입력한 후, WakaTime 대시보드에서 'Terminal' 프로젝트로 시간이 기록되는지 확인합니다.
